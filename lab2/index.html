3 : Meta tag
Metadata is information about data.Metadata will not be displayed on the page, but will be machine parsable.Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.
4 : Body tag
The body tag defines the document's body. The body element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.
5 : Section tag
The section tag defines sections in a document, such as chapters, headers, footers, or any other sections of the document.
6 :Div tag
The div tag is used to group block-elements to format them with CSS.
7: Aside tag
The HTML aside tag is an HTML 5 element that defines a section that is tangentially related to the content around it in the HTML document. This tag is also commonly referred to as the aside element.
8 : Nav tag
The nav tag defines a set of navigation links. It contains links to different parts of the web site. Notice that NOT all links of a document should be inside a nav element. The nav element is intended only for major block of navigation links.
